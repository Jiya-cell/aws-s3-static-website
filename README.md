# AWS S3 Static Website

My first hands-on AWS project, built while studying for AWS Cloud Practitioner.

## What I built
A static website hosted on Amazon S3 with public access enabled via a bucket policy.

## Steps taken
- Created an S3 bucket
- Enabled static website hosting
- Configured a bucket policy for public read access
- Deployed an HTML file

## Live demo
https://jiya-portfolio-2026.s3.eu-north-1.amazonaws.com/index.html
## What I learned
- S3 bucket creation and configuration
- Public access settings and bucket policies
- Static website hosting on AWS
